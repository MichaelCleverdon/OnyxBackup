General Information about Ruby

Ruby was made back in 1995 by a man named Yukihiro "Matz" Matsumoto by combining his favorite parts from Perl, Smalltalk, Eiffel, Ada, and Lisp. His overall intention with making the langauge was to "make Ruby feel natural, not simple"

Matz initially said that he wanted to make a scripting language that was "more object oriented than Python, and more powerful than Perl"

In Ruby, everything is an object (similar to Smalltalk). This allows for any variable, regardless of if it is a 'primitive type' or not, to have methods performed on them.

gcd.rb

#Ruby GCD program
#run with "ruby gcd.rb"

def gcd(num1, num2)
    while(num1 != num2)
        if(num1>num2)
            num1 = num1 - num2
        else
            num2 = num2 - num1
        end
    end
    num1
end

puts gcd(35, 49)

Output

7

primetest.rb

#Ruby prime number program
#run with "ruby primetest.rb"

def prime(num)
  isPrime = true
  for i in 2..(num-1) #for(int i = 2; i <= num-1; i++)
    if((num % i) == 0)
      isPrime = false
    end
  end
  isPrime
end

puts prime(31)

Output

true

Why use Ruby over other languages?

  • Ruby was made to be readable to humans first, computers second. This makes it easy to understand code without knowing everything about Ruby.
    • Want to repeat code a certain amount of times, use 5.times { puts someString }
    • String manipulation, no sweat. String are indexed: helloWorldString['World'] = 'WORLD' produces helloWORLD
    • Need to convert a string to a different type, just use .to_i or .to_f to convert the string to an integer or float respectively
  • Since it's so easy to read Ruby, it allows for collaboration to be a lot easier between team members. With less time being spent on explaining your code and how it works, the more work that can be done.
  • Ruby is still updated and maintained! Ruby 3 came out on Christmas of 2020 and 3.0.1 just came out on the 5th of April
  • Ruby can be used for creating Model-View-Controller(MVC) web applications using the Ruby on Rails framework. This creates more flexibility for programmers who have never made web applications before
    • Plenty of websites you use are designed with Ruby on Rails:
    • Hulu
    • Github
    • KickStarter
    • AirBnB
  • Ruby has exception handling features, that make it more enticing over other scripting programming languages
  • Garbage collection!!!
  • Ruby runs on most modern operating systems, and is highly portable
  • Ruby allows you to alter its essential parts. An example of this is allowing a programmer to rename the + operator on numbers to plus
